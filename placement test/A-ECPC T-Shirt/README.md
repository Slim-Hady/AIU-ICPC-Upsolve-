###  The Logic
You are given a color name

The rules are clearly stated in the problem:
1. If the input is **"Red"**, your program should output **"Contestant"**.
2. If the input is **"Green"**, your program should output **"Coach"**.
3. If the input is **"Blue"**, your program should output **"Judge"**.

### A Note for Beginners: Case Sensitivity
In programming contests, the "Judge" (the automated system checking your code) is extremely strict. 
* **Spelling matters:** "Contestant" is correct, "Contestants" is wrong.
* **Capitalization matters:** "Contestant" is correct, "contestant" (lowercase 'c') will result in a **Wrong Answer** verdict.
